I built this website using Hugo for static site generation. This allows me to update the content by writing Markdown and pushing it to GitHub where I have set up an action to automatically build the static pages.

For the looks, I based my CSS on PicoCSS. The icons used are from Font Awesome, and the text animation for headers uses TypeIt. I used the font Source Serif Pro from Google fonts for the headers.
