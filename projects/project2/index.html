Weather Forecasting Tool

A small Python tool for forecasting electric vehicle charging curves.

Overview

This project implements time-series forecasting for EV charging curves using scikit-learn.

Features

  • Handles short time series (≤ 40 samples).
  • Provides a simple interface for predictions.
  • Includes visualization of charging curves.

Usage

python forecast.py --input data.csv
